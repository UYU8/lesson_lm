设计一个支持push,pop,top 操作 并能在**常数时间O(1)**检索到最小元素的栈。

- 栈是什么？
  - FIlO 逻辑 一端操作
  - 线性数据结构 （受限的）
  - ADT 抽象数据类型
    存储结构 + 行为
    - 数组 简单 + 连续 new.Array().fill() 时间+连续（不是栈的行为 不推荐使用下标 因为栈只能在栈顶操作）
    - 链表 复杂 + 指针的开销 不连续 执行速度（new）

    - 栈的行为
      - push 入栈
      - pop 出栈
      - peek 取栈顶元素
      - isEmpty 判断栈是否为空
      - size 栈的大小
      - clear 清空栈
